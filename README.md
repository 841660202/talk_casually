# 纸聊

这个应用程序使用Google的Flutter移动框架开发，是一个实时聊天应用程序，为了能专注于APP设计，应用程序的服务端使用Googler的Firebase平台。程序程序的名称为纸聊，意为像传递小纸条一样的简约聊天，登录后所有好友都在，消息无缝同步，还能发送图片。支持Android、iOS等多个平台，多端消息漫游同步。*（由于使用了谷歌的后端服务，该应用程序必须在科学上网时才能正常使用。）*
